# Profile descriptions / Opisy profili

Full descriptions of each mode. These are intentionally kept **out of the startup menu** (which stays short) and surfaced here and in the in‑app Library.
Pełne opisy każdego trybu. Celowo trzymane **poza menu startowym** (które pozostaje krótkie) i pokazywane tutaj oraz w Bibliotece w aplikacji.

---

### 1 · Analysis / Analiza
**EN:** Diagnostic report only, with zero changes to the system. Result: you learn what is slowing the machine down.
**PL:** Wyłącznie raport diagnostyczny, bez żadnych zmian w systemie. Efekt: dowiadujesz się, co spowalnia komputer.

### 2 · Optimization / Optymalizacja
**EN:** The main mode. Final menu of tweaks, each with an effect description shown **before** you choose. Recommended.
**PL:** Tryb główny. Finalne menu tweaków, każdy z opisem efektu pokazanym **przed** wyborem. Polecane.

### 3 · Restore / Przywracanie
**EN:** Roll back settings from a previous optimization in a single choice.
**PL:** Przywróć ustawienia z poprzedniej optymalizacji jednym wyborem.

### 4 · Repair Windows / Naprawa i odbudowa Windows
**EN:** Audit plus basic or advanced repair of the system.
**PL:** Audyt oraz naprawa podstawowa lub zaawansowana systemu.

### 5 · Power plans / Kreator planów zasilania
**EN:** Power plans tuned to a goal (calibrated on real, verified plans), with backup/restore and factory reset.
**PL:** Plany zasilania pod cel (skalibrowane na realnych, sprawdzonych planach), z backupem/przywracaniem i resetem fabrycznym.

### 6 · Automation / Silnik automatyzacji
**EN:** Background rules: game → gaming plan, battery → eco plan, with a watchdog log.
**PL:** Reguły w tle: gra → plan gaming, bateria → eco, z logiem watchdoga.

### 7 · App packs / Paczki aplikacji (winget)
**EN:** Install a whole app set in one choice; export your own pack.
**PL:** Instalacja całego zestawu aplikacji jednym wyborem; eksport własnej paczki.

### 8 · Voice assistant / Asystent głosowy
**EN:** Experimental, offline. A few fixed commands.
**PL:** Eksperymentalny, offline. Kilka stałych komend.

### 9 · Library / Biblioteka
**EN:** Recipes (saved sets of tweaks) and full session history with rollback.
**PL:** Przepisy (zapisane zestawy tweaków) i pełna historia sesji z rollbackiem.

### 10 · Privacy & AI / Prywatność i AI
**EN:** Disable Copilot, Recall, telemetry, advertising ID and suggestions (reversible).
**PL:** Wyłącz Copilot, Recall, telemetrię, ID reklamowe i sugestie (odwracalne).

### 11 · Root‑cause / Analiza przyczyn
**EN:** Explains WHY the computer is slow (which service, driver, disk…).
**PL:** Tłumaczy DLACZEGO komputer jest wolny (która usługa, sterownik, dysk…).

### 12 · Report analysis / Analiza raportu
**EN:** Reads saved sessions and says in plain language: optimization SUCCESS / PARTIAL / FAILED (with %).
**PL:** Czyta zapisane sesje i mówi po ludzku: optymalizacja UDANA / CZĘŚCIOWA / NIEUDANA (z %).

### 13 · Debloat / Debloat
**EN:** A reversible, anticheat-safe background trim for gaming — **debloat, not core optimization**. Creates an automatic System Restore point, then stops a curated SAFE set of services and closes known background apps. Undo via [3] Restore. It never touches the game, anticheat, GPU driver or critical services. Tiers: [1] safe (default), [2] aggressive (opt-in, also closes launchers). Note: process count alone does not raise FPS — trimming heavy background load does.
**PL:** Odwracalne, bezpieczne dla anticheata odciążenie tła pod gry — **debloat, nie optymalizacja rdzenia**. Tworzy automatyczny punkt przywracania, potem zatrzymuje wyselekcjonowany BEZPIECZNY zestaw usług i zamyka znane aplikacje w tle. Cofnięcie przez [3] Przywróć. Nigdy nie rusza gry, anticheata, sterownika GPU ani usług krytycznych. Poziomy: [1] bezpieczny (domyślny), [2] agresywny (opcjonalny, zamyka też launchery). Uwaga: sama liczba procesów nie podnosi FPS — robi to ścięcie ciężkiego tła.
